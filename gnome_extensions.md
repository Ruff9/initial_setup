# Extensions Gnome

Une fois le paquet "vanilla-gnome-desktop" installé, sélectionner "Gnome" lors de l'ecran de connection pour avoir la bonne interface.

## Installées par défaut, à activer

- Ubuntu AppIndicators
- Ubuntu Docks (+ réglages)
- Ubuntu Tiling Assistant

## À ajouter

- Unite shell

Permet de supprimer les bandeaux de titre des fenêtres
Suivre instructions sur cette page : https://github.com/hardpixel/unite-shell
